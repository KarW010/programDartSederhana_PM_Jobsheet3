Program Cek Bilangan Prima ini merupakan implementasi kode modular dalam bahasa Dart yang memisahkan logika matematika, tampilan, dan alur kontrol utama ke dalam fungsi-fungsi terpisah untuk menjaga kebersihan kode dan kemudahan pemeliharaan. Seluruh proses pengembangan dilakukan menggunakan metodologi Git branching, di mana fungsi isPrima untuk logika pengecekan dan tampilkanHasil untuk iterasi output dikembangkan secara terisolasi pada branch masing-masing sebelum akhirnya digabungkan ke dalam branch utama (main). Pendekatan ini tidak hanya meminimalkan risiko merge conflict, tetapi juga menciptakan struktur proyek yang profesional dengan dokumentasi sistematis melalui file README.
